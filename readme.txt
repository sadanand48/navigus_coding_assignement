Flask App
Requirements:Python 3,flask
Steps to run
1.Go to codingassignment directory
2.run python3 app.py on terminal
once app is deployed on localhost,
http://127.0.0.1:5000/products: to show products
http://127.0.0.1:5000/search?name=RedBUll : to search a product
http://127.0.0.1:5000/favourites:To show favourites
Make a post request on http://127.0.0.1:5000/setfavourite to set a  favourites
